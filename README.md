EV Fleet Management System - Project bundle
Folders:
- backend: Flask app
- datasets: mock CSV datasets
- notebooks: starter Jupyter notebooks
- ml_models: saved model artifacts
- Dockerfile & docker-compose: for local deployment

To run locally:
1. python -m venv venv
2. source venv/bin/activate
3. pip install -r backend/requirements.txt
4. python backend/app.py
